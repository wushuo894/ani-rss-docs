# qBittorrent 设置

## 下载速度限制

单位：kiB/s。

用于覆盖当前任务在 qBittorrent 中的下载速度上限，适合在高峰期控制带宽占用，避免影响局域网中的其他设备。

## 上传速度限制

单位：kiB/s。

用于限制做种时的上传速度，适合带宽较小或希望保留更多上行给其他服务的场景。

## 分享率

"-1" 表示禁用，"-2" 使用全局设置

用于覆盖 qBittorrent 的单任务分享率规则。达到指定分享率后，任务是否停止做种仍以 qBittorrent 的相关策略为准。

## 总做种时长

"-1" 表示禁用，"-2" 使用全局设置

用于控制单个任务的最长做种时长，适合希望统一限制保种时间的场景。

## 非活跃时长

"-1" 表示禁用，"-2" 使用全局设置

当任务长时间无上传、无下载、无连接活动时，可按此阈值处理，适合自动清理长期无活跃度的任务。

## qB 保存路径

开启后将使用 qBittorrent 的保存路径配置 **（下载路径不受影响）**

通常用于兼容 qBittorrent 自己的临时目录、未完成目录或默认保存策略；但 ani-rss 中配置的下载位置仍然会继续参与实际任务路径控制。

::: warning
请务必在 qBittorrent 内开启并配置好临时下载位置，否则可能会导致下载位置错误
:::

::: tip
如果你已经在 ani-rss 中为不同订阅精细配置了保存目录，修改该选项前建议先用测试订阅验证一次最终下载路径是否符合预期。
:::
