# 页面设置

### 按星期展示

按星期展示，优先显示当前星期。

适合追更场景：开启后，番剧列表会更偏向按一周播出节奏来查看，今天所在的星期通常会被优先展示，方便快速查看今日更新。

### 显示评分

显示 [Bangumi](https://bgm.tv/) 评分。

适合在选番、补番时快速参考作品热度和口碑。如果某个条目尚未同步到 Bangumi，或 Bangumi 暂无评分，则该位置可能不会显示内容。

### 显示视频列表

显示视频列表按钮。

开启后，可以更方便地查看当前订阅或预览条目已经识别到的本地视频文件，适合排查以下问题：

- 是否已经下载过某一集
- 自动跳过为什么生效/未生效
- 本地文件是否已被正确识别

### 自定义 CSS

用于自定义网页界面的样式，例如：

- 调整卡片布局、字号、间距
- 隐藏不常用按钮
- 自定义亮色/暗色主题效果

你可以前往 <https://github.com/wushuo894/ani-rss-css> 复制现成的 `CSS` 模板后再按需修改。

::: tip 使用建议
自定义 CSS 只会影响页面显示效果，不会影响订阅、下载、重命名等实际功能逻辑。
:::
