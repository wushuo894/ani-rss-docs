# 代理设置

将用于 [Mikan](https://mikanani.me/)、[Bangumi](https://bgm.tv/)、[TMDB](https://www.themoviedb.org/)、[Telegram](https://telegram.org/)

::: warning
仅支持 HTTP 代理
:::

当你遇到以下问题时，通常建议优先检查或启用代理：

- RSS 获取超时
- SSL 握手异常
- DNS 解析失败
- Bangumi / TMDB / Telegram 无法访问

代理设置主要影响程序访问外部网站与 API 的请求过程，不会改变本地文件路径、下载器挂载路径等本地存储配置。

::: tip 适用场景
如果你所在网络环境对境外站点访问不稳定，或者使用默认运营商 DNS 时经常报错，代理通常会比单纯更换 DNS 更稳定。
:::
