开始开发 ​

1. 关闭本地正在运行的 ani-rss ​

INFO

如果没有端口冲突可以跳过这一步

2. 启动后端服务 ​

  • 用 IntelliJ IDEA 打开本项目。
  • 找到 /src/main/java/ani/rss/Main.java 并点击右上角的绿色调试按钮就好了。

3. 启动前端开发环境 ​

bash
# 进入前端目录
cd ui

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
